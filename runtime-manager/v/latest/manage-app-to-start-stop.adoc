= To Start and Stop a Mule Application
:keywords: runtime manager, anypoint platform, private cloud

You can start, restart, or stop a Mule application through the Runtime Manager UI.

To start or restart a Mule application:

[[navigate_to_runtime_manager_settings]]
include::manage-include-navigate-to-settings.adoc[]

. Select Start or Restart from the top-left dropdown menu in the panel.

To stop a Mule application:

. Select the application that you want to stop.
+
For more guidance with this step, see the instructions on starting a Mule application.
+
. Select Stop from the top-left dropdown menu in the panel.
