= Delete a Server

The DELETE operation deletes the specified server.

Example Request:

[source]
----
DELETE http://localhost:8080/console/api/servers/local$73d89173-290e-4cb3-a61c-e11deb74767d
----

Example Response:

[source]
----
200 OK
Server: Apache-Coyote/1.1
Date: Sun, 08 Nov 2009 00:12:55 GMT
----
