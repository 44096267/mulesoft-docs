= To Delete All Email from a Folder over IMAP(S)
:keywords: email, connector, configuration, imap
:toc:
:toc-title:

toc::[]

Anypoint Studio, Design Center: *Email* connector

[[short_description]]
Eliminates (expunges) from a mailbox folder all email that is flagged for deletion with *Mark As Deleted*.

[[expunge_folder]]
. After adding the *Email* connector to your flow, select the *Expunge Folder* operation.
. Set up or select an IMAP or IMAPS configuration for the operation (see link:email-imap-to-set-up[To Set Up an IMAP or IMAPS Email Configuration]).
. If you are deleting the file from a folder other than the inbox, provide the name of the folder, for example, `Trash`.
