= To Set Up an SFTP Configuration
:keywords: ftp, connector, matcher, directory, listener
:toc:
:toc-title:

toc::[]

Anypoint Studio, Design Center: FTP connector

[[short_description]]
An SFTP configuration is required for all SFTP operations with the FTP connector. 

. Open the `FTP Configuration` window.
+ You can open it from any FTP operation in your flow or from the Connections Explorer in Anypoint Studio.
. Provide a Name for the configuration. Example: `FTP configuration`.
. Provide connection settings:
  * Connection Type: SFTP Connection
  * Host domain name or server IP address.
  + Example: my.sftpsite.com
  * Port for the transport protocol. 
  +
  Defaults to `22`.
  * Password, if required by the host.
  * Passphrase for the Identity File (if provided).
  * Identity File location for a PKI private key (if needed).
. Provide a Working Directory, if needed.
+ 
Example: `my/working/dir`
. Provide the Preferred Authentication Methods (one of these): _TODO, WHAT'S THE DEFAULT?_
  * `gssapi_with_mic`
  * `public_key`
  * `keyboard_interactive`
  * `password`
. *Known Hosts File* to check whether the server key matches. Aborts if not.

For other configuration options, go to link:#see_also[See Also].
